SMS Trivia Tool

Send a text to +1 781 996 3571 to get music information.

Supported commands:

  • name bieber
  • bio radiohead
  • review born to run
  • album blood bank
  • song littlest birds

APIs Used:

  • Rovi for metadata
  • Twilio for SMS

The original version of this hack was created by Brian Fife in 24 hours at Music Hack Day Amsterdam 2012.

Note that google voice will support SMS responses, but Skype does not.
